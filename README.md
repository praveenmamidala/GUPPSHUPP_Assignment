Personalized AI Chat Assistant

This project is a personalized AI chat assistant that remembers user preferences and responds with different personalities. It uses Hugging Face LLMs to analyze chat history and generate replies that are more engaging and context-aware.

Features

Memory Extraction

Reads the last 30 user messages from a file.

Extracts structured information in JSON, including:

User preferences

Emotional patterns

Important facts for personalization

Personality Engine

Allows the assistant to respond in different personalities, such as:

Calm mentor

Witty friend

Therapist style

Makes responses more fun, empathetic, or thoughtful depending on the chosen persona.

Before / After Personality

You can see the difference between a neutral reply and a personalized reply with personality.

Demonstrates how memory and personality improve AI responses.

JSON Output

Saves memory and responses in final_output.json for easy reference or further processing.
